# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-15

### Added
- Complete CJ Affiliate API integration
- Product Search API (GraphQL) - Search advertiser product catalogs with affiliate tracking links
- Advertiser Lookup API (REST) - Search and discover advertisers with performance metrics
- Commission Detail API (GraphQL) - Real-time commission tracking
- Link Search API (REST) - Search affiliate links with tracking
- Program Terms API (GraphQL) - Access program commission rates
- Promotional Properties API (GraphQL) - Manage tracking PIDs
- Offer Feed API - Access automated offers
- Built-in file-based caching system with TTL support
- Comprehensive exception handling with CJException
- PSR-4 autoloading
- Full XML and JSON response support
- Guzzle HTTP client integration
- Complete documentation and 10+ usage examples
